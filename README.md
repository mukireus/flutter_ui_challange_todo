# To-Do
Bu uygulama eğitim için yapılmıştır. Düzenlemeler ve geliştirme fikirleri için bana ulaşabilirsiniz.

### Kurulum
```
git clone https://github.com/mukireus/flutter_ui_challange_todo.git
```
### Kullanımı
#### Menü
Eklemek için `+` iconuna tıklayınız. Grup ve görev ekledikten sonra listenecektir. Görevleri silmek için sağa kaydırmanız yeterli.
